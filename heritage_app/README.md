# 🌍 World Heritage Hub: A Living Encyclopedia of Human Culture

The **World Heritage Hub** is a mobile-responsive platform designed for communities worldwide to share, preserve, and interact with diverse aspects of human heritage. 

Managed by **Team Bees**, this platform organizes global knowledge into six core cultural pillars, validated by community custodians and elders.

---

## ✨ Key Features

### 🌎 Interactive Cultural Map
- **Full-Screen 3D Globe**: Built with `react-globe.gl` for a truly immersive exploration experience.
- **Emoji-Coded Pillars**: Instantly identify types of heritage (📖 Stories, 🎮 Games, 🌿 Medicine, etc.) via visual markers.
- **Geographic Origins**: Link every entry to its specific ethnic and geographic home.

### 📚 The Living Encyclopedia
- **Content-Rich Feed**: A premium, frosted-glass interface for browsing cultural entries.
- **Six Foundational Pillars**: 
  - **Stories & Oral Traditions**: Myths, legends, and proverbs.
  - **Traditional Games**: Digital adaptations and rules (e.g., Mancala, Ayo).
  - **Traditional Medicine**: Verified herbal remedies and healing practices.
  - **Cuisine**: Recipes and the cultural soul of food.
  - **Crafts**: Material culture, weaving, and metallurgy.
  - **Festivals**: A global calendar of rituals and celebrations.

### 🛡️ Community Validation & Trust
- **3+ Confirmation Rule**: Entries earn "Community Verified" status after three successful validations.
- **Elder Verification**: Gold-tier badges for entries endorsed by recognized cultural custodians.
- **Guardian Reputation**: Contributor tiers from "Newcomer" to "Legacy Guardian."

### 📊 Heritage Pulse Analytics
- **Real-Time Data**: Track total contributions and verification health.
- **Content Breakdown**: Visual analytics of growth across different cultural categories.

## 🛠️ Technical Stack

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **UI Architecture**: [Material UI (MUI) v6](https://mui.com/) + [Tailwind CSS v4](https://tailwindcss.com/)
- **Visuals**: [react-globe.gl](https://vasyl.me/react-globe.gl/), [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Frosted White Glassmorphism, Dark Slate aesthetic
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit)

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm, yarn, or pnpm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
